## Features

* Prime Number Calculator

## Installation through Composer

```shell
$ composer require alustau/math 
```

## Usage
It's easy to use, see to example below:

```php

use Math\PrimeNumber;

$math = new PrimeNumber();

$math->isPrime(11); //will return: true

```
